# 刑事申诉书

申诉人：周俊玲，女，汉族，1977年4月出生，工作单位：怀集县司法局，住广东省怀集县怀城街道碧桂园山语一街33幢508房，电话号码：13822654748.

申诉人是王博合同诈骗罪一案的受害人之一，申诉人对深圳市福田区人民法院作出的（2024）粤0304刑初1142号《刑事判决书》的判决结果不服（见证据一：《刑事判决书》）。一、申诉人认为法院对犯罪人王博认定的罪名无异，但认为法院对王博认定的犯罪事实不符，据以定罪量刑所适用的法律错误，量刑畸轻，申诉人对该裁判结果不服。二、申诉人认为公诉机关对案中起策划、组织作用的唐芷滢没有受到追诉，不服；对广州一航汽车服务有限公司的主管人员李焕铿没有受到追责，不服；王博和李焕铿均是以公司的名义签订的合同和协议，其所在公司及公司法定代表人没有受到追诉，不服；三、对法院和公诉机关违反法定程序办案，不服。现向检察院提出申诉。

# 事实与理由

一、法院对犯罪人王博所认定的犯罪事实不符，适用法律错误，导致量刑幅度把握错误，量刑畸轻。

（一）对犯罪人王博认定自首和认罚认罪的事实不符，适用法律错误。

（2024）粤0304刑初1142号《刑事判决书》第5页第5行至第8行认定犯罪人王博为“自首”和“认罪认罚”与事实不符，以此作为“减轻处罚”、“酌情从轻处罚”的理由，于法无据。

1.犯罪人王博的行为不属于自首。根据《刑法》第67条规定：“犯罪以后自动投案，如实供述自己的罪行的，是自首。对于自首的犯罪分子，可以从轻或者减轻处罚。其中，犯罪较轻的，可以免除处罚。被采取强制措施的犯罪嫌疑人、被告人和正在服刑的罪犯，如实供述司法机关还未掌握的本人其他罪行的，以自首论。犯罪嫌疑人虽不具有前两款规定的自首情节，但是如实供述自己罪行的，可以从轻处罚；因其如实供述自己罪行，避免特别严重后果发生的，可以减轻处罚。”“被告人王博是经公安机关传唤后主动到案接受调查”的（见证据一：《刑事判决书》第5页5行），不属于以上《刑法》第67条前两款规定的自首的情节，理由如下：

（1）王博不是自动投案。王博是经公案机关传唤后才到派出所接受调查的。也就是说王博的犯罪事实在传唤前已被公安机关发觉（此前已有受害人报案并提供了确凿的证据材料，公安机关已查明王博的主要犯罪事实），可见，王博是受到公安机关传唤才到公安机关的，而不是直接的、主动地到公安机关投案的，根据最高法《关于处理自首和立功具体应用法律若干问题的解释》第一条第一款规定：“自动投案：是指犯罪事实或者犯罪嫌疑人未被司法机关发觉，或者虽被发觉，但犯罪嫌疑人尚未受到讯问、未被采取强制措施时，主动、直接向公安机关、人民检察院或者人民法院投案。”因此，王博“经公安机关传唤后主动到案接受调查”的行为不属于《刑法》第一款规定“自动投案”的情节；王博所供述的罪行并非司法机关尚未掌握的罪行；因王博在传唤前已有受害人向派出所报案并提供了确凿的证据材料，公安机关已查明王博的主要犯罪事实，因此，王博的情况并非属于“如实供述司法机关还未掌握的本人其他罪行的”的情况；因此，王博的行为不属于“自动投案”的情节；（2）本案也并非因王博如实供述自己的罪行，从而避免了特别严重后果发生，因为在其供述时一切皆已发生；（3）没有自动投案，在办案机关调查谈话、讯问、采取调查措施或者强制措施期间，即使被告人王博如实交代办案机关掌握的线索所针对的事实，也不能认定为自首。以上事实与理由证实王博不具备《刑法》第67条规定的自首的情节，不能以自首论处。既然不是自首，就不能以此作为减轻处罚的依据。

2.法院认定的认罪认罚与事实不符：（1）认罪方面：王博在的庭审现场上对控诉机关所指控的罪名不服。还认为这不是犯罪，不是诈骗，还辩解说滴滴公司尚这般操作，后是经法官解释教育才促使其认罪的，可见王博对自身行为尚未有犯罪意识，既然没有犯罪意识又何来悔罪心理？（2）

认罚方面：A.王博家属只分别退赔被害人唐芷滢和陈明军仅仅5000元，这是极少的数目，这个数目与所诈骗他们俩的数额135910元相差甚远，并未体现积极退赔的态度，且俩被害人也均未对被告人王博有谅解的意思，重要的是：申诉人作为被害人之一，被诈骗数额就已是25万元（未计利息)，属于数额特别巨大，王博至今分毫未退赔，可见王博根本就没有认罚态度，悔罪表现；B.王博的行为带给申诉人及其家庭严重的伤害，申诉人作为受害人对王博是恨之入骨的，相信其他受害人也如此，绝不可谅解。如果在宣告前，王博能将所诈骗的钱财全额退还，主动缴纳罚金，方可认定为认罚，方可认定为《刑事诉讼法》第十五条规定的“愿意接受处罚”。因此本案《刑事判决书》第5页第7行法院认定王博为认罪认罚与事实不符，适用法律错误，依法不得减轻、从轻处罚。B.犯罪情节特别严重：犯罪人王博犯罪手段狡猾：王博以高息为诱饵、以免费使用车辆为幌子收取押金，让被害人产生错误认识处分自己的财产，以先履行合同约定的前两个月利息为手段，诱使对方当事人继续签订和履行合同，在此期间还隐瞒事实真相、篡改合同期限，最后达到骗取几个被害人全部押金的目的，所骗金额特别巨大，并挥霍一空，而且被害人均为不特定的人，没有一定的手段都不可能骗到这么多人，可见王博手段恶劣，情节特别严重，不应当酌情从轻处罚；C.犯罪人王博犯罪的行为社会危害大，不宜酌情从轻处罚。王博为集合犯。在此之前，有据可寻的已背负7宗民间借贷案（证据二：中国执行信息公开网查询结果和判决书），王博严然已是妥妥的“背债专业户”，而这7宗案件中，其中张健与王博的民间借贷案[见证据二中的（2021）粤0303民初35794号《民事判决书》]，王博的作案方法就跟本案相同，事实上是一宗刑事案件，只是原告当时没有以刑事案向有权侦查的机关报案而已，也就是说如此类型的案件王博已非初犯，可见王博的行为社会危害性大，故，不宜酌情从轻处罚；从案卷可见王博所诈骗的对象是四个不特定主体一一被骗的四个人互不认识，按广州2023年人均纯收入为49327元（有两个爱害人是广州人），而按照肇庆市人均纯收入为32562元（申诉人是肇庆人)，对于一个普通家庭来说，被骗6位数字的钱财，将危害到一个家庭经济的正常运转，王博危害了4个家庭经济几年的正常运转，可见王博的行为社会危害性极大，故，不宜酌情从轻处罚；D.王博再犯的可能性极大。以上7宗民间借贷案中，其中有2宗为“有履行能力而拒不履行生效法律文书确定义务”，案号分别为：

（2020）粤0303执2553号和（2020）粤0306执32030号（见以上证据二），一宗为违反财务报告制度，以上情况王博被法院列为失信被执行人名单，王博的行为已构成拒不执罪，可见，犯罪人王博的劣根性顽固，再犯的可能性极大，酌情从轻处罚，难以让其从根本上得到改造，服刑时间越短，再犯的可能性越大。王博这样的罪犯于法于理于情都不应当“酌情从轻处罚”。E.王博犯罪主观恶意性极大。王博从本案中就已骗得53万多元既没退还给被害人也没有清偿旧债务，而是挥霍一空或转移财产，可见王博犯罪的主观恶意性极大，并不适合酌情从轻处罚。

综上，王博的行为不具有自首情节，不属于自首，不具有减轻处罚的法定情节；王博没有根本性的认罪认罚的表现，且主观恶意性大，情节特别严重，其行为社会危害性大，不具有从轻从宽处罚的依据。因此，综合本案情况，判决书第5页第5行至第8行，王博行为以自首和认罪认罚论处与事实不符，认定“减轻处罚”“从轻处罚”适用法律错误。

（二）对犯罪人王博量刑幅度把握错误，量刑起点认定错定，也是导致本案对王博量刑畸轻，罪责刑不一致的一个重要的原因。

1.犯罪人王博诈骗数额达535910元（尚未退赔525910元），属于数额特别巨大，法院只判处二年二个月，与法定刑不一致。根据《刑法》第二百六十六条规定：“诈骗公私财物，数额较大的，处三年以下有期徒刑、拘役或者管制，并处或者单处罚金；数额巨大或者有其他严重情节的，处三年以上十年以下有期徒刑，并处罚金；数额特别巨大或者有其他特别严重情节的，处十年以上有期徒刑或者无期徒刑，并处罚金或者没收财产。”和根据《最高人民法院、最高人民检察院关于办理诈骗刑事案件具体应用法律若干问题的解释》第一条规定：“诈骗公私财物价值三千元至一万元以上、三万元至十万元以上、五十万元以上的，应当分别认定为刑法第二百六十六条规定的‘数额较大’、‘数额巨大’、‘数额特别巨大’。”以及根据2021年6月17日最高法、最高检《关于常见犯罪的量刑指导意见（试行）》“1.构成合同诈骗罪的，根据下列情形在相应的幅度内确定量刑起点：

（1）达到数额较大起点的，在一年以下有期徒刑、拘役幅度内确定量刑起点。（2）达到数额巨大起点或者有其他严重情节的，在三年至四年有期徒刑幅度内确定量刑起点。（3）达到数额特别巨大起点或者有其他特别严重情节的，在十年至十二年有期徒刑幅度内确定量刑起点。依法应当判处无期徒刑的除外。2.在量刑起点的基础上，根据合同诈骗数额等其他影响犯罪构成的犯罪事实增加刑罚量，确定基准刑...”。本案王博诈骗数额已达53万多元，根据以上规定，已完全达到“数额特别巨大”的起点，应当“在十年至十二年有期徒刑幅度内确定量刑起点”，在量刑起点的基础上，根据合同诈骗数额等其他影响犯罪构成的犯罪事实应当增加刑罚量，确定基准刑。而判决书第5页第3行以“数额巨大”为起点，判处了“二年二个月”，即是“在三年至四年有期徒刑幅度内确定量刑起点”的基础上进行减轻、从轻处罚后所判处的刑期，明显与事实严重不符，量刑起点幅度认定错误，造成了量刑畸轻，罪责与法定刑不一致的结果。

2.在判决后申诉人提出申诉，按法院要求请求主审法官出具判后答疑时，询问本案主审法官为何如此轻判，并拿出以上规定作为对抗的理由，但法官认为本案审的是合同诈骗罪，以上《刑法》第二百六十六条是诈骗罪的规定。申诉人认为在本案的审理中明显是法官没有正确处理诈骗罪与其他特殊诈骗罪的关系。《刑法》第二百六十六条规定的诈骗罪条款与《刑法》第二百二十四条规定的合同诈骗罪条款是普通法条与特别法条的关系，从以上规定看，最高院、最高检《关于办理诈骗刑事案件具体应用法律若干问题的解释》规定的意思是：办理合同诈骗罪、集资诈骗罪、信用证诈骗罪等特殊诈骗罪时对“数额较大”、“数额巨大”、“数额特别巨大”，分别认定为《刑法》第二百六十六条规定诈骗公私财物价值“三千元至一万元以上”、“三万元至十万元以上”、“五十万元以上”。如果不是这样的理解的话，那诈骗数额五十万以上，主审法官以“数额巨大”（见判决书第5页第3行）为罪责量刑的依据又是什么呢？当然该主审法官对以上辩解不肯书面出具说明。

（三）这样的判决结果没法达到预防犯罪的目的，没有法律效果和社会效果，有违《刑法》第二条的规定。

王博所犯的合同诈骗罪诈骗数额达53万多元，在没有法定的减轻、从轻处罚的情况下，只判决二年二个月，这个刑罚结果远远低于社会评价的平均值，难以让广大民众接受。监狱有吃有住，还有公费医疗，如此轻判，如此般低的犯罪成本，只会让人感觉犯罪不可怕，怕的是骗不到“巨款”，只会让不良之徒，穷尽心思挖掘“发财”的途径，这样势必助长犯罪的社会歪风。刑罚的目的之一是预防人们犯罪（包括重新犯罪），如果犯罪得不到应有的惩罚，那就根本实现不了刑罚的目的，根本实现不了善治的社会效果和法律效果，有违《刑法》第二条和第二条的规定。别说本人对这样的判决不服，即使广大民众都不服。

二、公诉机关对王博所在的单位犯罪漏审漏诉。

# 事实与理由：

（一）李焕鏗涉疑构成另一案的合同诈骗罪。

李焕铿作为公司直接负责的主管人员以支付初期的2个月的租金，开始履行合同的方法，诱骗申诉人重新签订合同延长代管代租期限（见以上证据十八、十九），以获取车辆其余全部租金的收益，在协议约定期间内无正当理由拒不支付其余租金，又不愿意解除协议退还车辆，直到租期满后一个月仍占用车辆不肯归还，致使申诉人既拿不到租金又找不到人找不到车，直到最后车辆被派出所强制扣押（见以上证据二十九：申诉人与李焕铿的聊天记录）。至目前为止两辆车尚欠租金11000元，数额较大。根据《刑法》第二百二十四条规定：“有下列情形之一，以非法占有为目的，在签订、履行合同过程中，骗取对方当事人财物，数额较大的，处三年以下有期徒刑或者拘役，并处或者单处罚金；数额巨大或者有其他严重情节的，处三年以上十年以下有期徒刑，并处罚金；数额特别巨大或者有其他特别严重情节的，处十年以上有期徒刑或者无期徒刑，并处罚金或者没有财产：....(三)没有实际履行能力、以先履行小额合同或者部分履行合同的方法，诱骗对方当事人继续签订和履行合同的；（四）收受对方当事人给付的货物、货款、预付款或者担保物后逃匿的....”因此，李焕铿的行为构成了合同诈骗罪。李焕铿还有可能以同样的方法与唐芷滢的朋友陈明军签订这样的协议，因此，李焕铿构成多宗合同诈骗罪。

（二）因李焕铿是以其所在公司的名义签订的协议，根据《刑法》第二百三十一条规定，李焕铿成立合同诈骗罪，则其所在公司也以该罪追责，并且涉疑多宗这样的合同诈骗罪，应当追究其所在公司及公司法定代表人的法律责任（见证据三十一：广州市一航汽车服务有限公司的工商企业登记信息）。

（三）唐芷滢涉疑构成的李焕铿合同诈骗罪的共犯。

六、判决不明确。《刑事判决书》第5页倒数第5行“二、责令被告人王博向被害人唐芷滢、陈明军、周俊玲共同退赔375910元”，该判行对每个被害人各退赔分别为多少，不明确，须知道被害人三人不是经济共同体，其中两人还互不认识，这样的判决在日后的执行过程中必定会引起纠纷，法院应当审理清楚王博向各受害人退赔的具体金额，并明确地在判决书中载明（若本案能重审，申诉人将附带民事诉讼）。

七、公诉机关违反审查起诉的程序规定。根据《刑事诉讼法》第一百七十条规定：“人民检察院审查案件，应当讯问犯罪嫌疑人，听取辩护人或者值班律师、被害人及其诉讼代理人的意见，并记录在案。辩护人或者值班律师、被害人及其诉讼代理人提出书面见的，应当附卷。犯罪嫌疑人认罪认罚的，人民检察院应当告知其享有的诉讼权利和认罪认罚的法律规定，听取犯罪嫌疑人、辩护人或者值班律师、被害人及其诉讼代理人对下列事项的意见，并记录在案：（一）涉嫌的犯罪事实、罪名及适用的法律规定；（二）从轻、减轻或者免除处罚等从宽处罚的建议；（三）认罪认罚后案件审理适用的程序；（四）其他需要听取意见的事项.. 。在本案审查起诉阶段申诉人并没有接到检察院的任何通知，没有听取申诉人作为被害人的意见，如果听取被害人的意见，今天也许不会这么折腾。另外，公诉机关提出的幅度刑量刑建议与罪责刑严重不相符。

八、法院办案程序违法。本案在2024年9月29日开庭，2024年9月30日已宣判，过了大半个月也没见判决书，于是打电话给法院，主审法官要求申诉人书面申请才可领取。最后申诉人再次打电话以路途遥远为由，要求法院寄送判决书才肯寄给申诉人（见证据三十四：开庭通知和宣判通知运单凭证），本案为定期宣告的案件，根据《刑事诉讼法》第二百零二条的规定，应当在宣告后立即将判决送达给当事人，法院违反该法律程序。当时收到判决书的时候正好有病无法做好上诉工作（见证据三十五：诊断证据)，否则不会让判决书放任生效。法院没有及时送达判决书，导致申诉人在法定期限内没有及时上诉。

九、（1）法院违反法定程序拒给付收件凭证。甲诉人由于不可抗力没法及时上诉，改为申诉，便向法院寄送申诉材料，但收到申诉人的申诉材料后没有给申诉人出具收件凭证（见证据三十六：向法院寄送申诉材料运单凭证），这就很难证明申诉人已提交和没提交哪些证据材料，这往往成为法院以当事人没有提交足够的证据材料而不予以立案。根据《最高人民法院关于人民法院立案工作的暂行规定》第十条规定：“人民法院收到诉状和有关证据，应当进行登记，并向原告或者-自诉人出具收据，收据中应当注明证据名称、原件或复制件、收到时间、份数和页数，由负责审查起诉的审判人员和原告、自诉人签名或者盖章。对于不予立案或者原告、自诉人在立案前撤回起诉的，应当将起诉材料退还，并由当事人签收。”法院违反该规定。（2）申诉人提出申诉后，法院要求当事人向主审法官或向信访部门出具书面的“判后答疑”没有法律的依据（见证据三十七：《民事案件再审申请所需材料清单》及运单凭证），本案主审法官不愿意出具判后答疑，导致申诉人没法顺利地向法院提出重审，如果需要法院应当由立案庭通知主审法官出具，而不是由当事人求他们出具。申诉人提交的是《刑事申诉状》（见以上证据三十六），法院却叫人补充民案材料，不得不说法官工作很儿戏，当然不排除是专业问题，更不排除欺负当事人不懂法。

综上，根据《中华人民共和国刑事诉讼法》第二百五十一条规定：“当事人及法定代理人、近亲属，对已经发生法律效力的判决、裁定，可以向人民法院或者人民检察院提出申诉，但是不停止判决、裁定的执行。”和根据《人民检察院办理刑事申诉案件规定》第二条、第八条第五款和第十二条的规定，申诉人对王博的畸轻裁判、对判决书的裁判不明确、对唐芷滢和李焕铿漏审漏诉、对王博和李焕铿所在公司及其法定代表人漏审漏诉、公诉机关和法院违反法定程序办案，申诉人不服。现依法提出申诉，恳请检察院重新审查、提起公诉。
